
# Mac Keyboard Symbol List:
⌘ is command
⌥ is option
⌃ is control
⇧ is shift
⇪ is caps lock
← is left arrow
→ is right arrow
↑ is up arrow
↓ is down arrow
⇥ is tab
⇤ is backtab
↩ is return
⌤ is enter
⌫ is delete
⌦ is forward delete
⇞ is page up
⇟ is page down
↖ is home
↘ is end
⌧ is clear
␣ is space
⎋ is escape
⏏ is eject
